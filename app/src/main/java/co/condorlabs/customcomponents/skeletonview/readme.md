# Skeleton fragment

Show a skeleton fragment for a view 

## Basic Usage

Create a new instance of the fragment

```
    SkeletonFragment.newInstance("Completion")
```

## Argument

| Name | Description  |
| -| - |
|  type  | Type of the skeleton to show. Available: `Completion`  |
